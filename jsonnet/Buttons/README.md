按键修改说明
----------------

本文件是对键盘按键功能进行自定义修改的说明。

键盘按键功能设置位于 `jsonnet/Buttons/` 文件夹下。其中包括以下文件：
- README.md：本说明文件。
- Layout26：定义了 26 键中文和英文键盘的各按键功能。
- LayoutNumeric9：定义了9键形式数字键盘的各按键功能。
- Toolbar：定义了工具栏和浮动面板中各按键功能。
- Common：定义了一些通用按键功能。

按键的具体配置语法可以参考 `doc/按键语法说明.md`。

修改步骤：
1. 关闭本 README.md 文件。
2. 打开同一文件夹下需要修改的按键文件，例如 `Layout26.libsonnet`。
3. 根据需要修改对应按键的功能配置，修改完后保存文件。
4. 回到元书皮肤界面，长按皮肤，选择「运行 main.jsonnet」生效。
